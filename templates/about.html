All movie data comes from IMDB's Non-Commercial Datasets, a set of publicly available TSV files that IMDB publishes and updates daily. The following six dataset files are downloaded and merged to build the local database:

  • title.basics â€” title, original title, release year, runtime, genres, and adult flag
  • title.ratings â€” IMDB rating and number of votes per title
  • title.crew â€” director and writer IDs for each title
  • title.principals â€” top-billed cast members per title
  • name.basics â€” person names (used to resolve director, writer, and cast IDs to names)
  • title.akas â€” alternative titles with language and country information

Only feature-length movies that have at least one rating are included. The database is refreshed periodically by re-downloading all six files and rebuilding the local SQLite database, so ratings, vote counts, and newly added titles stay up to date.
